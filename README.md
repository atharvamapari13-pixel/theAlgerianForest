# 🔥 Algerian Forest Fire Prediction Web App

A Machine Learning-powered web application that predicts whether a forest area is **Fire** or **Not Fire** based on environmental and meteorological conditions. The model is trained on the **Algerian Forest Fires Dataset** and deployed using Flask.

---

## 📌 Project Overview

Forest fires are a major environmental hazard. This project leverages historical weather and forest data to build a classification model that can predict the likelihood of a fire outbreak.

The application provides a simple web interface where users can input parameters like temperature, humidity, wind speed, etc., and get an instant prediction.

---

## 🎯 Objective

- Predict forest fire occurrence (**Fire / Not Fire**)
- Build an end-to-end ML pipeline
- Deploy the model using Flask
- Provide real-time predictions via UI

---

## 📊 Dataset Information

- Dataset: Algerian Forest Fires Dataset
- Features include:
  - Temperature
  - Relative Humidity (RH)
  - Wind Speed (Ws)
  - Rain
  - FFMC, DMC, DC, ISI (Fire Weather Index components)
